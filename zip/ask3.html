!===============================
ΣΥΝΑΡΤΗΣΗ Αξιολόγηση(α, β): ΑΚΕΡΑΙΑ
ΜΕΤΑΒΛΗΤΕΣ
	ΑΚΕΡΑΙΕΣ: α, β, ω
ΑΡΧΗ
	α ← α + 2
	β ← β – 3
	ω ← α * β – 2
	Αξιολόγηση ← ω ^ 2 – (α + β)

ΤΕΛΟΣ_ΔΙΑΔΙΚΑΣΙΑΣ

Μετατρέψτε τη συνάρτηση σε διαδικασία συμπληρώνοντας τα κενά. Αυτή τη φορά, για να το κάνετε, θα πρέπει να συμπληρώσετε τα κενά με το πληκτρολόγιο σας.

Όταν είστε έτοιμοι πατήστε το κουμπί ΕΛΕΓΧΟΣ


Σε περίπτωση που δυσκολεύεστε, μην διστάσετε να πατήσετε το κουμπί "ΒΟΗΘΕΙΑ" όπου θα εμφανιστούν πληροφορίες που θα βοηθήσουν να λύθεί η άσκηση.

    Στοιχείο 1ο: Ένα υποπρόγραμμα πρέπει να καλεστεί στο κυρίως πρόγραμμα για τνα χρησιμοποιηθεί.

    Στοιχείο 2ο: Οι μεταβλητές που αλλάζουν μέσα σε μια συνάρτηση δεν επιστρέφονται στο κυρίως πρόγραμμα. Αντίθετα, μια διαδικασία τις επιστρέφει.

    Στοιχείο 3ο: Για να επιστρέψουν οι μεταβλητές α και β αναλλοίωτες στο πρόγραμμα, θα πρέπει να χρησιμοποιηθούν οι μεταβλητές τοπ1 και τοπ2.
