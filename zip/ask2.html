Η προηγούμενη άσκηση ήταν μια από τις περιπτώσεις που μπορεί μια διαδικασία να μετατραπεί σε συνάρτηση.


Οι διαδικασίες μπορούν να εκτελέσουν οποιαδήποτε λειτουργία από αυτές που μπορεί να εκτελέσει ένα πρόγραμμα. Να εισάγουν δεδομένα, να εκτελέσουν υπολογισμούς, να μεταβάλλουν τις τιμές των μεταβλητών και να τυπώσουν αποτελέσματα. Με τη χρήση των παραμέτρων αυτές τις τιμές μπορούν να τις μεταφέρουν και στα άλλα υποπρογράμματα. Συνεπώς, μπορούν να εκτελέσουν και οποιαδήποτε λειτουργία μιας συνάρτησης.


Μετατρέψτε την διαδικασία σε συνάρτηση. Για να το κάνετε, σύρετε τις λέξεις και συμπληρώστε τα κενά της συνάρτησης.

Όταν είστε έτοιμοι πατήστε το κουμπί ΕΛΕΓΧΟΣ


Σε περίπτωση που δυσκολεύεστε, μην διστάσετε να πατήσετε το κουμπί "ΒΟΗΘΕΙΑ" όπου θα εμφανιστούν πληροφορίες που θα βοηθήσουν να λύθεί η άσκηση.

    Στοιχείο 1ο: Η απάντηση θα πρέπει να αποθηκευτεί σε ξεχωριστή μεταβλητή και όχι στο όνομα της διαδικασίας, όπως θα γινόταν σε μια συνάρτηση.

    Στοιχείο 2ο:Η απάντηση δεν μπορεί να αποθηκευτεί στις μεταβλητές α,΄β ή γ καθώς είναι ακέραιες μεταβλητές και οι απαντήσεις είναι λέξεις.

    Στοιχείο 3ο:Η απάντηση θα πρέπει να αποθηκευτεί στη μεταβλητή 'απ'.
